# Catudy Flutter App

This is the final Catudy mobile client.

The current slice is intentionally a skeleton:

- Material 3 Flutter app shell
- `go_router` route foundation
- Riverpod bootstrap
- Catudy Site color tokens
- Catudy logo and mascot assets
- bottom navigation placeholders for Home, Stats, Calendar, Pet Room, and Profile

The Unity project remains in `client/UnityProject` as a legacy prototype only.

## Commands

```powershell
flutter pub get
flutter test
flutter run
```

If Flutter is not on PATH, run it through the local SDK path:

```powershell
C:\Users\arca\flutter\bin\flutter.bat test
```
